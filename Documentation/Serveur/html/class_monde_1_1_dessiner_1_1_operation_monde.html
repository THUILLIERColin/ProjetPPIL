Description détaillée

La classe OperationMonde permet de faire des opérations sur le monde, comme par exemple le changement de repère, la transformation en monde écran, etc.

Cette méthode permet de créer un monde à partir de vecteur de coordonnées

Paramètres
bgMpoint bas gauche monde
hdMpoint haut droit monde
bgEpoint bas gauche ecran
hdEpoint haut droit ecran

Cette méthode sert à créer un objet de la classe GenererMonde qui représente la transformation entre le système de coordonnées monde (qui représente les objets que l'on veut dessiner) et le système de coordonnées écran (qui représente la fenêtre de dessin).
